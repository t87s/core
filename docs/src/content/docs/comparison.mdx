## vs Raw Redis

| Aspect | Raw Redis | t87s |
|--------|-----------|------|
| Invalidation | Manual `DEL` calls | Declarative `invalidates` |
| Type safety | Stringly-typed keys | Typed `defineTags()` |
| Prefix matching | Manual implementation | Built-in |
| Staleness detection | None | Cloud adapter |

## vs BentoCache

| Aspect | BentoCache | t87s |
|--------|------------|------|
| Tags | String-based | Type-safe factories |
| Invalidation | Imperative `deleteByTag()` | Declarative return value |
| Prefix matching | ❌ | ✅ |
| Observability | Basic | Intelligence layer |

## When to Choose t87s

- You have **hierarchical data** (users → posts → comments)
- You want **type-safe** cache keys
- You're tired of **forgetting to invalidate** related cache
- You want to **know when cache is stale**
